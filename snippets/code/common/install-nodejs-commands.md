=== "Ubuntu"
    ```
    curl -sL https://deb.nodesource.com/setup_15.x | sudo -E bash -

    sudo apt install -y nodejs
    ```

=== "MacOS"
    ```
    # You can use homebrew (https://docs.brew.sh/Installation)
    brew install node

    # Or you can use nvm (https://github.com/nvm-sh/nvm)
    nvm install node
    ```
